# dependency

## 更新记录
### 2021年5月8日 - 1.0.0.2
1、修复展开\收拢所有节点卡死的问题  
2、增加打开文件校验，只允许打开PE文件  
3、支持拖动文件  

### 2021年5月10日 - 1.1.0.3
1、导出函数表添加右键菜单，支持复制函数名和查询函数
2、修复bug
